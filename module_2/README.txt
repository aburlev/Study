Задача проекта - выполнить анализ предложенных данных, провести их очистку, выявить зависимости,
и на их основе отобрать данные для построения модели.

Предствленные данные - это набор сведений об учениках, таких как пол, возраст, семья, образование и 
работа родителей, быт, времяпрепровождение и итоговый бал по математике. 
Именно зависимость итоговой успеваемости по математике от социально-бытовых факторов необходимо
выяснить.

Проведен анализ данных, устранены выбросы, заполнены пропущенные значения.
Проведен корреляционный анализ по численным и номинативным параметрам. 

Отобраны параметры для построения модели.
